# Simulation Engine

1. `npm install`

2. Need Postgressql DB

3. Need to create surreal database in local

4. add .env in the project root directory
    ```
    DB_USER=postgres
    DB_PASSWORD=madhav@1993
    DB_HOST=localhost
    DB_PORT=5432
    DB_DATABASE=surreal

5. To Run simulation engine in local instance
    `npm start`
